# AJ Miller T. Perez — Portfolio

A single-page portfolio site for AJ Miller T. Perez (Product & Program Leader · Engineering Quality · AI-Enabled Transformation), built with React + Vite.

## Tech

- React 18
- Vite 5

## Getting started

```bash
npm install      # install dependencies
npm run dev      # start the dev server
npm run build    # build for production (outputs to dist/)
npm run preview  # preview the production build locally
```

## Structure

- `index.html` — page shell and metadata
- `src/main.jsx` — React entry point
- `src/App.jsx` — the full single-page app
- `src/index.css` — production stylesheet
- `public/favicon.svg` — brand mark favicon
